# Machine-Test-API
## APIs

## Category APIs:

# Post- Add Category
http://localhost:8080/api/categories

# Get- Get all categories
http://localhost:8080/api/categories

# Get- Get Category by Id
http://localhost:8080/api/categories/{categoryId}

# Put- Update Category by Id
http://localhost:8080/api/categories/{categoryId}

# Delete- Delete Category by Id
http://localhost:8080/api/categories/{categoryId}
